Complete Customer Service Chatbot System - Replit Project Prompt
Project Overview
Build a comprehensive customer service chatbot system with the following components:

Embeddable chat widget for WordPress websites
Admin dashboard for managing chat visibility and settings
Multi-channel message routing (Email, Slack, Trello, APIs)
ChatGPT integration for automated responses
Representative mobile/web interface
Voice-to-text messaging capabilities
Core Requirements
1. Embeddable Chat Widget
Create a JavaScript widget that can be embedded on any WordPress site
Position: Bottom-right corner, non-intrusive overlay
Should NOT interfere with existing website layout
Responsive design for mobile and desktop
Easy one-line embed code: <script src="your-domain.com/chatbot.js"></script>
Customizable colors and branding
Smooth animations for open/close states
2. Admin Dashboard (Web Interface)
Page Management:

Interface to select which pages/URLs the chatbot appears on
Whitelist/blacklist functionality
WordPress page detection and selection
Preview mode to test widget placement
Settings Panel:

Customize widget appearance (colors, position, size)
Configure business hours
Set automated responses and triggers
Manage representative availability status
3. Message Routing & Integrations
Email Integration:

Send all conversations to specified email addresses
Create email threads for ongoing conversations
Email notifications for new messages
Slack Integration:

Route messages to designated Slack channels
Real-time notifications for new chats
Allow responses from Slack back to customers
Trello Integration:

Create Trello cards for each conversation
Update cards with conversation progress
Custom fields for customer information
API Webhooks:

Configure multiple API endpoints for message payloads
Customizable webhook formats (JSON, XML)
Retry logic and error handling
Real-time and batch sending options
4. ChatGPT Integration
Integrate OpenAI API for automated responses
Smart handoff to human representatives
Customizable AI personality and knowledge base
Fallback responses when AI is uncertain
Context awareness for multi-turn conversations
Representative override capabilities
5. Representative Interface
Web Dashboard:

Real-time chat interface with multiple conversation threads
Customer information panel
Chat history and analytics
File sharing capabilities
Typing indicators and read receipts
Quick response templates
Mobile App Features:

Cross-platform (iOS/Android) using React Native or Flutter
Push notifications for new messages
Voice-to-text integration using device speech recognition
Photo upload for representative profiles
Offline message queuing
Dark/light mode toggle
Representative Management:

Profile creation with name and photo upload
Online/offline status management
Shift scheduling and availability
Performance metrics and chat volume tracking
6. Voice & Media Features
Voice message recording and playback
Voice-to-text conversion for representatives
Image and file sharing
Screen sharing capabilities (web interface)
Audio/video call integration (optional)
